Mis Proyectos Destacados

Proyecto: Landing Page para Startup "EcoGrow"

Diseño UI/UX y Desarrollo Frontend.

Creación de una landing page moderna y responsive para una startup de productos sostenibles. Enfoque en la experiencia de usuario intuitiva y la estética visual que comunica los valores de la marca.

  • Herramientas: Figma, HTML5, CSS3 (Sass), JavaScript.
  • Desafío: Optimización para conversión y adaptabilidad móvil.
  • Resultado: Incremento del 25% en registros durante la fase beta.

Scar0k: Diseño que conecta.

Proyecto: Aplicación Móvil "FitJourney"

Diseño de Interfaz y Experiencia de Usuario (UI/UX).

Diseño completo de una aplicación de seguimiento de fitness. Se desarrolló un flujo de usuario intuitivo y una interfaz visualmente atractiva para motivar a los usuarios a mantenerse activos.

  • Fases: Investigación, Wireframing, Prototipado, Testing de Usabilidad.
  • Plataformas: iOS & Android (diseño unificado).
  • Características clave: Seguimiento de actividades, planes personalizados, comunidad.

Scar0k: Experiencias móviles memorables.

Proyecto: Rebranding Completo para "Café del Sol"

Identidad Visual y Diseño Web.

Desarrollo de una nueva identidad de marca, incluyendo logotipo, paleta de colores, tipografía y un sitio web corporativo para una cafetería local. El objetivo era modernizar su imagen y atraer a un público más joven.

  • Servicios: Diseño de Logo, Manual de Marca, Diseño Web Responsivo.
  • Impacto: Aumento del reconocimiento de marca y tráfico al sitio.
  • Concepto: Fusión de tradición y modernidad.

Scar0k: Marcas que inspiran.

Proyecto: E-commerce para Boutique de Joyería "Gema Única"

Diseño y Optimización de Tienda Online.

Diseño de una plataforma de e-commerce elegante y segura para una boutique de joyería. Se puso énfasis en la presentación visual de los productos y un proceso de compra simplificado.

  • Plataforma: Shopify (Personalización del tema).
  • Funcionalidades: Carrito de compras, pasarelas de pago, galería de productos.
  • Mejoras: Navegación intuitiva y diseño visual atractivo para resaltar las joyas.
